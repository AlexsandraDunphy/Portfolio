Minhas Habilidades

  • HTML

    HTML é uma linguagem de marcação de hipertexto utilizada na criação de documentos e páginas da web. Os marcadores, chamados de “tags”, servem para indicar a função de cada elemento na página, como textos, imagens e vídeos.

  • CSS

    É uma linguagem de estilo que controla a aparência e o layout dos elementos em uma página da web.

  • JavaScrip

    JavaScript é uma linguagem de programação usada para adicionar interatividade e dinamismo às páginas da web. Ela permite manipular elementos HTML, responder a eventos, validar formulários, fazer requisições a servidores e muito mais.

  • API

    API significa Application Programming Interface (Interface de Programação de Aplicação) é um conjunto de regras que permite a comunicação entre diferentes softwares

  • Jira Software e Confluence

    É uma plataforma de gerenciamento de projetos e rastreamento de problemas, enquanto o Confluence é uma plataforma de colaboração e documentações de equipes.

  • Cypress

    Cypress é uma ferramenta de teste automatizado para desenvolvimento web. Permite criar testes interativos que simulam as ações do usuário em um navegador, facilitando a garantia de qualidade e o desempenho das aplicações.
